Questão 1

A função do arquivo index.js deve ser: (1) Criar os eventos para inserção, remoção, etc; (2) Invocar as funções do módulo lista.js para adicionar, remover, concluir e listar itens; (3) Usar as informações fornecidas pelo módulo lista.js para montar uma tabela com todos os itens da lista de compras na tela.

As funções acima devem utilizar os dados atuais de localStorage, ou seja, salvar, remover e listar devem acessar o armazenamento. Ao recarregarmos a página, todos os itens de compra também devem ser carregados automaticamente (evento "load").
